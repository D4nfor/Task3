# Проект модуля триангуляции

## Описание
Проект реализует модуль для триангуляции трехмерных моделей с использованием простейших алгоритмов.

## Архитектура проекта

### Основные классы:

1. **BaseModel** - абстрактный базовый класс для всех моделей
    - Содержит список вершин
    - Предоставляет базовые методы работы с вершинами
    - Определяет абстрактные методы для реализации в наследниках

2. **TriangulatedModel** - класс триангулированной модели
    - Наследуется от BaseModel
    - Добавляет список треугольников
    - Реализует методы для работы с триангуляцией

3. **Vertex** - класс для представления вершины в 3D
    - Хранит координаты x, y, z
    - Предоставляет методы для вычисления расстояний

4. **Triangle** - класс для представления треугольника
    - Содержит три вершины
    - Предоставляет методы для вычисления площади и периметра

5. **Triangulator** - утилитный класс для триангуляции
    - Реализует простейшие алгоритмы триангуляции
    - Fan triangulation (триангуляция веером)
    - Простая триангуляция полигонов

6. **Main** - демонстрационный класс
    - Показывает работу всех компонентов
    - Демонстрирует полиморфизм и наследование

## Алгоритмы триангуляции

### 1. Fan Triangulation (Триангуляция веером)
- Берется первая вершина как центральная
- Создаются треугольники: (центральная, i, i+1)
- Простейший метод для выпуклых многоугольников

### 2. Polygon Triangulation
- Для выпуклых многоугольников
- Создаются треугольники от первой вершины к каждой паре соседних

### 3. Point Set Triangulation
- Для произвольного набора точек
- Создаются треугольники из каждых трех последовательных точек

## Использование

```java
// Создание базовой модели
BaseModel model = new BaseModel("Модель") {
    @Override
    public void displayInfo() { /* реализация */ }
    
    @Override
    public double calculatePerimeter() { /* реализация */ }
};

// Добавление вершин
model.addVertex(0, 0, 0);
model.addVertex(1, 0, 0);
model.addVertex(0, 1, 0);

// Триангуляция
TriangulatedModel triangulated = Triangulator.triangulateFan(model);

// Работа с результатом
triangulated.displayInfo();
triangulated.printTriangles();
```
## Особенности реализации
1. Наследование: TriangulatedModel наследует от BaseModel
2. Инкапсуляция: Все поля классов приватные с публичными геттерами
3. Полиморфизм: Абстрактные методы в BaseModel переопределяются в наследниках
4. Расширяемость: Легко добавить новые алгоритмы триангуляции

## Компиляция и запуск
```bash
  javac geometry/*.java
  java geometry.Main
```
## Тестирование

В проекте реализованы модульные тесты с использованием **JUnit 5**.

Тесты находятся в директории `test/` и проверяют основные компоненты:
- `Vertex`
- `Triangle`
- `TriangulatedModel`
- `Triangulator`

Для запуска тестов в IntelliJ IDEA необходимо пометить папку `test`
как **Test Sources Root** и запустить нужный тест.
